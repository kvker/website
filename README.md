# 污斑兔的个人网站

这是我的个人网站项目，包含博客、应用展示等功能。

## 项目结构

```
my-website/
├── public/                 # 静态资源目录
│   ├── apps/              # 应用集合
│   │   ├── index.html     # 应用列表页面
│   │   └── todos/         # TODOS应用
│   ├── blogs/             # 博客文章
│   ├── styles/            # 样式文件
│   ├── scripts/           # 脚本文件
│   └── images/            # 图片资源
├── views/                 # EJS模板文件
├── routes/                # 路由文件
├── app.js                 # Express应用主文件
└── package.json           # 项目依赖配置
```
